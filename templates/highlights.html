Our free block explorer is different, here are some of the things you can do here that you won't find elsewhere:

  • Confirmation Predictions - We're the only site that can predict the probability that a transaction will confirm.
  • Fee Estimates - Get detailed network stats on how large a transaction fee you need to pay to be included quickly in a block.
  • Upload Public Metadata to the Blockchain (beta) - Upload a public key-value string of text to blockcypher's servers that will be viewable on the block explorer and via API. Here are examples of this uploaded data on a bitcoin transaction, address, & block.
  • Embeddable Widgets - Copy-paste a short HTML snippet and embed blockchain info into your website. Great for showcasing how many donations you've gotten.
  • Fastest Most Recent Transactions - See transactions long before they appear in other block explorers.
  • Open Source - See how this works, run your own version of the site, and submit pull requests.

We also have a number of cool tools like push tx & decoderawtransaction.
